# college-vue
## Installation Instructions

Follow these steps to install the application:
## Clone repository
```

git clone
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
